## Chinamap Panel Plugin for Grafana

基于Echarts2开发

![Chinamap](./src/imgs/chinamap-china.png)


## 时间序列数据作为数据源

支持数据源:

- Elasticsearch


包括的位置数据:

- 省份 
- 城市（暂时不支持展示）
